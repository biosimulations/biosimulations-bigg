BioSimulations-BiGG documentation
=================================

BioSimulations-BiGG provides a command-line application for publishing the `BiGG model repository <http://bigg.ucsd.edu/>`_ of flux balance models to the `BioSimulations <https://biosimulations.org>`_ repository for simulation projects.

This command-line program is run weekly by the GitHub action in this repository.
