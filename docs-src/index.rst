BioSimulations-BiGG documentation
=================================

BioSimulations-BiGG provides a command-line application for importing the `BiGG model repository <http://bigg.ucsd.edu/>`_ of flux balance models into the `BioSimulations <https://biosimulations.org>`_ repository for simulation projects.

This command-line program is run weekly by the GitHub action in this repository.
