1. Installation instructions¶

1.1. Requirements¶

  • Python >= 3.7

  • pip

1.2. Installing the latest revision from GitHub¶

After installing Python (>= 3.7) and pip, run the following command to install BioSimulations-BiGG:

pip install git+https://github.com/biosimulations/biosimulations_bigg#egg=biosimulations-bigg
